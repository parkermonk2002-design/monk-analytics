# Monk Analytics Website

A responsive, one-page static website for Monk Analytics.

## Files
- `index.html` — website structure and content
- `styles.css` — layout, colors, and responsive styling
- `script.js` — mobile menu and contact-form behavior

## Preview locally
Open `index.html` in a browser.

## Before publishing
1. Replace `hello@monkanalytics.com` with your real email address.
2. Replace sample portfolio cards with screenshots of your own projects.
3. Connect the form to Formspree, HubSpot, Netlify Forms, or another form service.
4. Confirm pricing and service descriptions.
5. Add a privacy policy if you collect visitor information.
